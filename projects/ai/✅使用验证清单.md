## 🚨 常见问题快速解决

### 问题：代理服务器启动失败
**解决：**
```bash
1. 检查Node.js: node --version
2. 检查端口占用: netstat -ano | findstr :3000
3. 以管理员身份运行启动脚本
```

### 问题：测试页面显示错误
**解决：**
```bash
1. 确保代理服务器正在运行
2. 检查网络连接
3. 验证API密钥是否有效
```

### 问题：主应用仍显示CORS错误
**解决：**
```bash
1. 确保访问 http://localhost:3000 (不是file://)
2. 手动点击"代理模式"按钮
3. 刷新页面重新初始化
```
