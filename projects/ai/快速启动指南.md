# 🚀 AI编程助手 - 快速启动指南

## 🎯 想要最佳体验？选择全血模式！

### 方法1：Node.js代理（推荐）
```bash
1. 双击 start-proxy.bat
2. 浏览器访问 http://localhost:3000
3. 点击"代理模式"按钮
4. 开始对话！
```

### 方法2：Python代理（备选）
```bash
1. 双击 start-python-proxy.bat
2. 浏览器访问 http://localhost:3000
3. 点击"代理模式"按钮
4. 开始对话！
```

### 方法3：直接使用（模拟模式）
```bash
1. 双击 index.html
2. 自动启用模拟模式
3. 开始对话！
```
